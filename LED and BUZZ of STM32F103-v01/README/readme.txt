

--------------------------------------------------蜂鸣器报警和LED照明节点--------------------------------------------------

//STM32F103最小系统板
//对  LED  和  蜂鸣器  的控制
//通过蓝牙串口 和STM32F407中转站  交互数据
//蓝牙组网，该组通信波特率位：19200
//需要使用一对蓝牙


板载LED：PC13


硬件连接：

	蓝牙串口：
		USART1_TX   	  PA.9
		USART1_RX	  PA.10
		
	LED控制引脚:	PC14（8种状态可调）
	
	
	蜂鸣器报警：  PB14
